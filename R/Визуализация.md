### Стандартные средства
- `plot()` - стандартная функция для создания простейших графиков.
**примеры:**
```R
z <- seq(-10, 10, 0.2)
# создаем вектор со значениями от -10 до 10 с шагом 0,2
dev.copy(jpeg, 'practice.jpeg')
# копирут графическое содержимое на указанное устройство 
# (jpeg - устройство, 'practice.jpeg' - название файла в который будет сохраненно)
plot(z, z**2, type = 'l', xlab = 'x', ylab = 'f(x)', col = 'blue')
# z и z**2 данные но оси х и у, type - тип графика, 
# xlab и ylab - названия осей ординат, col - цвет
lines(z, z**3, col = 'red')
# функция которая добавляет на существующий график еще одну линию 
# с данными х - z и y - z**3, col - соответственно цвет
legend(-3, 100,
       c('x^2', 'x^3'),
       lwd = c(1, 1),
       col = c('blue', 'red'),
       cex = 0.7)
# функция которая добавляет легенду для графиков
dev.off()
# функция которая указывает окончания информация о графическом содержание 
# которое необходимо сохоанить в файл
```

flats <- read.csv('flats.csv')

- `hist()` - функция предназначена для создания гистограммы. Лучше всего подходят для количественных данных. Гистограмма позволяет: упорядочить значения по возрастанию, разбить числовой отрезок на интервалы равной длины, посчитать, сколько значений попадает в каждый интервал, отметить полученные частоты в виде столбцов соответствующей высоты.
**примеры:**
```R
mycars <- mtcars

dev.copy(jpeg, 'practice2.jpeg')
hist(mycars$hp, main = 'Мощность двигателя', col = 'yellow', 
     border = 'red', xlab = 'Число лошадинных сил')
# border - цвет разделителя между колонками 
abline(v = mean(mycars$hp), col = 'green', lty = 2, lwd = 2)
abline(v = median(mycars$hp), col = 'blue', lty = 3, lwd = 2)
legend(250, 10,
       c('Среднее', 'Медиана'), 
       lty = c(2, 3),
       lwd = c(1, 1),
       col = c('green', 'blue'),
       cex = 0.7)
dev.off()
```

- `barplot()` - функция предназначена для создания столбчатой диаграммы. Для каждого уникального значения в наборе данных определить его частоту, отметить полученные частоты в виде столбцов соответствующей высоты.
**примеры:**
```R
cols <- rep(c("red", "blue"), 5)
cols <- c(cols, "red")
# создания списка с цветами
dev.copy(jpeg, 'practice2_barplot.jpeg')
barplot(tab, main = "Sectors", las = 2, cex.names = 0.65, 
        col = cols, ylab = "counts")
# las - развернуть надписи на оси х
# cex.names - размер надписей (по дефолту = 1)
dev.off()
```

-  `pie()` - функция предназначенная для создания круговой диаграммы.
**примеры:**
```R
labs <- names(tab)
# labs - список названий категорий
proc <- round(tab / sum(tab) * 100, 2)
proc_final <- paste0(proc, '%')
# proc_final - процент количества всех категорий
fill <- c("thistle1", "plum1", "palevioletred3", "maroon4", 
            "purple2", "paleturquoise",
            "cornflowerblue", "lightcyan3", "royalblue1",
            "seagreen2", "navy")
# fill - список цветов
pie(tab, main = 'S&P500', labels = proc_final, col = fill, cex = 0.7)
# labels - название долей
dev.copy(jpeg, 'practice2_piet.jpeg')
legend(x = -1.5, y = -0.8, labs, cex = 0.5, fill = fill,
       ncol = 3, bty = "n")
# ncol - расположить все названия в 3 колоеках
# bty - для каждой категории определить квадрат с цветом согласно fill
dev.off()
```
