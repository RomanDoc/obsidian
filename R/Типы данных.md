- `numeric` - тип данных для хранения любых чисел, как целых так и дробных
- `integer` - тип данных для хранения только целых чисел
- `logical` - логический тип данных `TRUE` и `FALSE`
- `character` - тип данных для хранения строковых значений
- `factor` - тип данных который записывается как целое число, но по сути таким не является. Определяется как метка на что-либо
